## Overview

Speak for Me connects to the following services:

| Service | Purpose | Required |
|---------|---------|----------|
| **Slack** | Core platform for message monitoring and suggestion delivery | Yes |
| **Anthropic (Claude AI)** | AI-powered suggestion generation | Yes (server-side) |
| **Stripe** | Payment processing and subscription management | For paid plans |
| **Google Sheets** | Weekly report export to spreadsheets | Optional |

The Slack integration is the foundation of Speak for Me. All other integrations extend the core functionality with billing, AI generation, and data export capabilities.
