  <what-built>
Complete web portal with:
- Slack OAuth authentication
- Dashboard with learning summary
- Style settings page (tone, formality, phrases, guidance)
- Conversations management page
- People context management page
- AI learning/feedback history page
- Weekly reports settings page
  </what-built>
  <how-to-verify>
1. **Start the dev server:**
   ```bash
   npm run dev --workspace=web-portal
   ```
   Portal should be available at http://localhost:3001

2. **Test authentication (PORTAL-01):**
   - Visit http://localhost:3001
   - Should redirect to /login
   - Click "Sign in with Slack"
   - Complete OAuth flow
   - Should redirect to dashboard

3. **Test dashboard (PORTAL-02):**
   - View learning summary showing personalization status
   - Verify stats cards show correct counts
   - Verify sidebar navigation works

4. **Test style settings (PORTAL-06, PORTAL-07):**
   - Navigate to Style Settings
   - Set tone and formality
   - Add phrases to use/avoid
   - Add custom guidance
   - Save and verify toast appears
   - Refresh and verify settings persisted

5. **Test conversations (PORTAL-08, PORTAL-09):**
   - Navigate to Conversations
   - If channels exist, verify they're listed
   - Test remove button with confirmation
   - Verify empty state shows /watch instructions

6. **Test people (PORTAL-05):**
   - Navigate to People
   - Click Add Person
   - Enter a Slack user ID and context
   - Save and verify in list
   - Test edit and delete

7. **Test AI learning (PORTAL-02, PORTAL-03, PORTAL-04):**
   - Navigate to AI Learning
   - Verify stats and refinement patterns
   - If refinements exist, expand to see details

8. **Test reports (PORTAL-10):**
   - Navigate to Reports
   - Toggle schedule on
   - Configure day, time, timezone
   - Select format and sections
   - Save and verify toast
   - Refresh and verify persisted

9. **Test sign out:**
   - Click sign out in sidebar
   - Should redirect to login
   - Should not be able to access dashboard

  </how-to-verify>
